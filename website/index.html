Version 0.1

Focused on YouTube import

Extension popup

Open NotebookTools on any YouTube video tab, choose a NotebookLM notebook, or create a new one, then import the current video.

In-page button

A white NotebookLM button appears beside YouTube's action bar so you can import without leaving the watch page.

More capture features are planned for a later release.

How it works

Three steps

  1. 01

    Sign in to NotebookLM

    Open NotebookLM once in Chrome so your session is active.

  2. 02

    Watch a YouTube video

    Go to any YouTube watch, Shorts, or embed page supported by the extension.

  3. 03

    Import to a notebook

    Click the in-page NotebookLM button or use the extension popup to send the video.

Install

Chrome Web Store submission

Version 0.1 is packaged in the repository under version 0.1/extension. Zip that folder to upload in the Chrome Developer Dashboard, or load it unpacked for testing.

View on GitHub
  1. Open chrome://extensions
  2. Enable Developer mode
  3. Click Load unpacked
  4. Select the version 0.1/extension folder
  5. Pin NotebookTools and open a YouTube video to test
