Features

Built for focused research workflows

YouTube import

Import the active video into any NotebookLM notebook, or create a new one from the extension popup.

In-page YouTube button

A NotebookLM control appears beside YouTube's action bar so you can import without opening the popup.

Selected text capture

Highlight text on any page, right-click, and save the selection to a local notebook.

Full webpage capture

Save the entire active page to a local notebook when you need a fallback outside NotebookLM.

How it works

Three ways to add a source

  1. 01

    On YouTube

    Click the in-page NotebookLM button or open the popup and choose a notebook.

  2. 02

    On any webpage

    Select text, right-click, and choose Add selection to NotebookTools.

  3. 03

    From the popup

    Pick an existing local notebook or type a new name, then add the full page.

NotebookLM import uses your signed-in browser session. Open NotebookLM first if notebooks do not appear.

Install

Load the extension in Chrome

NotebookTools is distributed as an unpacked developer extension. Clone the repo, then load the notebooktools-extension folder in Chrome.

Clone from GitHub
  1. Open chrome://extensions
  2. Enable Developer mode
  3. Click Load unpacked
  4. Select the notebooktools-extension folder
  5. Pin NotebookTools to your toolbar for quick access
